Скрипт парсит ссылки на все товары отдельно взятого продавца в Ozon.
Для обхода Cloudflare используется библиотека Silenium, для имитации действий реального пользователя.
Если скрипт натыкается на Cloudflare и его не пускают на сайт, то он пытается спарсить нужную страницу пока не получится.

Для корректной работы скрипта необходимо:
1) Установить все заимствования командой: "pip install -r requirements.txt";
2) Изменить в файле "constant.py" значение для TOTAL_PAGES (поставить число 
всех страниц каталога нужного продавца);   
3) Изменить в файле "constant.py" значение для BRAND_URL (поставить cсылку на первую 
страницу с товарами нужного продавца);
4) Запустить скрипт командой "main.py";
5) Скрипт спарсит все html страницы и достанет из них ссылки на товары продавца;
6) Появится файл 'links.txt', в который запишутся все полученные ссылки.
